src/main/java Chứa mã nguồn Java: servlet, controller, service,… sẽ được biên dịch vào WEB-INF/classes. src/main/resources Chứa các tài nguyên đi kèm mã Java: file cấu hình (.properties, .yml), template, thông điệp đa ngôn ngữ,… Khi build, Gradle tự động copy vào classpath. src/main/webapp Chứa tài nguyên web tĩnh và cấu hình web: HTML, CSS, JS, ảnh, JSP, cùng thư mục WEB-INF (với web.xml). Khi đóng gói, nội dung ở đây sẽ thành gốc của ứng dụng web.
